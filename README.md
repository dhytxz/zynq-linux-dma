# zynq-linux-dma
A user space DMA example on Zynq platform (ZC706) under Petalinux.
## Environment Requirement
### EDK Versions
Vivado 2014.4 with corresponding Petalinux.
### Hardware Platform
ZC706 Evaluation Board.
